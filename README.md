# Greeting Picture Switch

Простое приложение на C# Windows Forms с вводом имени и переключением изображений.

## Функциональность

- Ввод имени в текстовое поле
- Кнопка "button1" — выводит приветствие и показывает pictureBox1
- Клик по pictureBox1 — меняет текст на прощание, скрывает первую картинку и показывает pictureBox2

## Интерфейс

- textBox1 — поле для ввода имени
- button1 — кнопка приветствия
- label2 — отображение текста (приветствие/прощание)
- pictureBox1 — первая картинка (видима после приветствия)
- pictureBox2 — вторая картинка (появляется после клика)

## Пример работы

1. Пользователь вводит имя в textBox1
2. Нажимает кнопку → label2 показывает "Привет, Имя", появляется pictureBox1
3. Клик по pictureBox1 → label2 показывает "Прощай, Имя", pictureBox1 скрывается, pictureBox2 появляется

## Примечание

Для работы программы необходимы две картинки в pictureBox1 и pictureBox2. В текущем коде они предполагаются уже добавленными в дизайнере.
